My projects

Podcast System

A podcast system i made while studying database management systems

A PostgreSQL database for storing and manipulating data about podcasts, hosts and users
This project was made in accordance to ACID principles to ensure reliable and secure transactions
It is also quite effective at data storing, retrival, and management and is quite scalable while maintaining high performance
Airline Ticket System

An Airline Ticket System group project i was a part of while studying object oriented programming

This was a PostgreSQL database connected to Java via JDBC to manage flights, users and thier tickets
It was made in accordance to SOLID principles and utilized design patterns such as singleton, builder and factory
Despite some limitations that could be fixed, it is an effective system
Matrix Row Reduction Program

A program for solving systems of linear equations

A C++ program made to help solving problems in linear algebra
It uses Gaussian and Gauss-Jordan elimination to simplify the given matrix to its reduced row echelon form
It is capable of handling matrixes of different sizes of matrixes to be applicable in various problems
However i must note that it wasn't quite time effective, which was solved by using C++ instead of Java
